认识“SEO”

发表于 2017-05-13

1.SEO是什么？

　　Search Engine Optimization，搜索引擎优化。
　　对网站进行内部及外部的调整优化，改进网站在搜索引擎中关键词的自然排名，获得更多的展现量，吸引更多目标客户点击访问网站，从而达到互联网营销及品牌建设的目标。

2.SEO与网站优化

网站结构布局优化

  1. 控制首页链数量，不能太多，不能太少
  2. 扁平化目录层次，尽量跳转三次就能到达
  3. 导航SEO优化，尽量文字方式
  4. 使用面包屑导航，可以让用户了解当前所处的位置，了解网站的组织情况

代码优化

  1. 提高页面响应速度
  2. 语义化
  3. 该加title的加上
  4. <strong> 、 <em> 灵活运用
  5. HTML里把主要代码放前面
  6. 重要内容不放在JS里
  7. 不用&nbsp

3.人工智能与SEO

　　比如在传统排名算法计算出相关页面后，人工智能算法挑出与已知作弊页面有相同访问特征和其它特征的页面，降低其排名或者干脆不返回。
　　这对黑帽SEO是一个致命的打击
