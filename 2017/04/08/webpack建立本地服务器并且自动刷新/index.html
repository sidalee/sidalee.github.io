webpack建立本地服务器并且自动刷新

发表于 2017-04-08

webpack有什么用？

　　webpack将一切都当成模块，然后require引入，把其中不是js的通过loader解释一下，然后打包到一个js文件，html页面在script引入它，从而减少了http的请求次数，达到优化性能的目的

如何配置webpack服务器和热更新呢？

  1. 安装本地服务器：
    npm install webpack-dev-server --save-dev

  2. 在webpack.config.js里配置端口号等

  3. 在index.html里添加：
    <script src="http://localhost:9999/webpack-dev-server.js">

  4. 运行时指定根目录：
    webpack-dev-server --content-base public/

然后就大功告成了！可以在浏览器里看到我们的index.html，并且修改后不用刷新，保存就能看到效果！
