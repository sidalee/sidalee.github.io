微信分享接口开发

发表于 2017-07-02

  1. 引入微信api：

    • <script src="http://res.wx.qq.com/open/js/jweixin-1.0.0.js"></script>
    • 后台申请签名，具体步骤：
      1. 用户创建时间戳，随机字符串，当前需要分享的页面的url三个变量
      2. 将自己的appid和APPsecret作为请求参数获取access_token
      3. 根据access_token获取jsapi_ticket
      4. 将获取的jsapi_ticket，以及自己创建的三个变量进行签名
