JS之闭包该怎么理解？

发表于 2017-01-13

1.什么是闭包？

　　闭包是就是函数中的函数，里面的函数可以访问外面函数的变量，外面的变量的是这个内部函数的一部分。

2.闭包的两个用途

  • 使用闭包可以访问函数中的变量
  • 让变量值始终保持在内存中

　　第二点这么理解，我们的垃圾回收前面说过，js在使用完一个变量后会自动清除内存，而我们的闭包内的变量，虽然是用完了，但是她还依存着外部的全局变量，全局变量不会被清除，所以闭包内的局部变量也得以保存

3.闭包的应用场景

  1. 使用闭包代替全局变量（在立即执行函数里var一个变量，外面访问不到）
  2. 函数外或在其他函数中访问某一函数内部的参数
  3. 在函数执行之前为要执行的函数提供具体参数
  4. 在函数执行之前为函数提供只有在函数执行或引用时才能知道的具体参数
  5. 为节点循环绑定click事件，在事件函数中使用当次循环的值或节点，而不是最后一次循环的值或节点
  6. 暂停执行
  7. 封装私有变量
